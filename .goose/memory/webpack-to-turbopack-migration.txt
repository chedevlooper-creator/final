# webpack turbopack migration nextjs build-tooling
MIGRATION TASK: Remove Webpack configuration and migrate to Next.js modern tooling (Turbopack). Key requirements: 1) Remove next.config.js (webpack config) and webpack.config.js, 2) Remove webpack loaders from package.json, 3) Create next.config.ts (TypeScript) with Turbopack enabled, 4) Update scripts with --turbo flag, 5) Use SWC compiler (default), 6) Use next/image and next/font, 7) Enable tree shaking and performance optimizations.
