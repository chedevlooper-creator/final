src/lib/inventory/inventory-service.ts - Complete inventory service layer with 7 services:
- InventoryCategoryService: CRUD for categories (create, getById, getAll, update, delete)
- InventoryItemService: CRUD for items with search, filtering, auto-code generation (ITM-YYYY-SEQ)
- WarehouseService: CRUD for warehouses with location support
- WarehouseLocationService: CRUD for warehouse locations (zones, aisles, shelves, bins)
- InventoryStockService: Stock queries and management (getByItem, getByWarehouse, updateQuantity)
- InventoryMovementService: Movement operations (receipt, issue, transfer) with auto-code (MOV-YYYY-SEQ)
- StockAlertService: Alert management (create, checkLowStock, getAlerts, resolve)

All services use Supabase client, proper error handling, TypeScript type safety, support for filtering/pagination/search, and stock quantity updates on movements.
