## Ne yaparım

- Next.js App Router API route'u yazarım
- Supabase client entegrasyonu eklerim
- Authentication kontrolü yaparım
- Type-safe response dönerim
- Error handling ile ErrorHandler kullanırım
- Turkish error mesajları eklerim

## Ne zaman kullanılır

Yeni API endpoint'i gerekli olduğunda:
- Veri CRUD işlemleri
- Custom endpoint'ler
- Webhook handler'lar
- Cron jobs
- Batch operations

## Kullanım örneği

"Toplantılar API route'u oluştur" → /api/meetings
"MERNIS doğrulama endpoint'i" → /api/mernis/verify
"Export endpoint'i" → /api/export/donations
